About

What is this?

This website was created both as a personal development project and as a contribution to the NeoVim community.
Our aim is to showcase and track the rapidly evolving ecosystem of NeoVim plugins. Given the vibrant and expansive nature of the community, capturing every innovation is a quite challeing endeavor. We still strive to bring some unique insight into the plugin landscape.

Where do you get your data?

All the data is scrapped from Github via the offical API and stored in a private DB. I plan to open-source the code for this in the close future.

What did you use to build this?

From my few interactions with JavaScript, i took a inital disliking to it, so i decided to write this project in the next easiest language, Python. To be specific the frontend is build in Flask + HTMX, while the backend is pure Flask.
